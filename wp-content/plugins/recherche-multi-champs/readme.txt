Créer vos propres champs pour vos articles/pages et proposer une recherche basée sur ces champs à vos visiteurs.

== Description ==

Boostez les champs personnalisés de Wordpress avec ce plugin:<br><br>
- Création simplifiée des champs lors de la rédaction de vos articles et pages. <br>
- Activez le widget pour proposer à vos visiteurs la recherche sur vos champs.<br>
- Customisez l'affiche du résultat de la recherche dans les options du plugin.<br>
- Insérez le shortcode dans votre contenu pour afficher les champs.<br>
- Insérer un formulaire de recherche dans vos pages ou articles.<br><br>
<a href="https://www.crealion.net/demo01/portfolio/" target="_blank">Site démo</a>


== Installation ==

Pour installer le plugin: <br><br>
- Décompresser le fichier ZIP,<br>
- Placer le dossier décompréssé dans le dossier wp-content/plugins/ de votre site,<br>
- Dans l'administration de votre site, allez dans "Extensions" puis cliquer sur "Activer" pour le plugin "Recherche Multi-champs",<br>
- Un nouveau menu "Recherche Multi-Champs" apparaîtra alors.<br>
