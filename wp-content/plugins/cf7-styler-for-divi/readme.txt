Design your Contact Form 7 forms with Divi Visual Builder. No coding required.

== Description ==
Contact Form 7 Styler for Divi Visual Page Builder. Now you can Customize and design beautiful contact forms with Divi Visual Builder and without writing a single line of code.

[Check the Demo](http://divipeople.com/products/contact-form-7-styler)

[Divi](https://www.elegantthemes.com/gallery/divi/) WordPress Theme or Visual Page Builder and [Contact Form 7](https://wordpress.org/plugins/contact-form-7/) are required for this plugin.

### Features
* Easy Contact form 7 pickup
* By click design Input / Textarea/ Checkbox / Radio / Acceptance ( No Coding Required )
* You can design also success and error message
* You can set multiple column fields in contact Form 7 ( This can be done by using a wrapper class. The class you’ll be using for this is: “dvppl-cf7-col”) 


== Installation ==
Note: This plugin works with Divi Theme or Divi Builder and Contact Form 7. Make sure you have Divi Theme or Divi Builder and Contact Form 7 installed.

1. Upload the plugin folder to the /wp-content/plugins/ directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the ‘Plugins’ screen in WordPress
